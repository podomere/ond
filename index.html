Welcome to the OND GitHub Pages.

Please feel free to check out any Open Network Designs via the gh-pages branch which contains the designs. Please install git and then you can initialise a git repository on your local machine, then:

$ cd your_repo_root/ond
$ git fetch origin
$ git checkout gh-pages

If you need help with Git please spend some time here: GitHub help and/or quickly learn or get a feel for it here: Try Git
