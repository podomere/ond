About

Welcome to this Public Domain repository of network designs which are made freely available to download, use, share, fork, and remix in any way you see fit.

What next?

Please feel free to check out or fork the repository via the gh-pages default branch (which contains the designs and auto-generated web pages). First install git and then initialise a git repository on your local machine and clone this repository!

$ cd your_new_directory
$ git init
$ git clone https://github.com/podomere/ond.git

If you need help with Git please spend some time here: GitHub help and/or quickly learn or get a feel for it here: Try Git. Open Network Designs are written in markdown and more details are in the README
